# PK InfoSec — Sprint 1 MVP

Livrable visible immédiat avec 4 pages:
- Home (`index.html`)
- Services (`services.html`)
- About (`about.html`)
- Contact (`contact.html`)

## Aperçu rapide en local

Option 1 (Python):
```bash
python3 -m http.server 8080
```
Puis ouvrir: http://localhost:8080

Option 2 (Node):
```bash
npx serve .
```
